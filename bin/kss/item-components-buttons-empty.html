1.1.2 Empty

Styles for button with class .btn--empty, is used by empty buttons (with no content) or input radios, like slider controls or size pickers.

Examples

Default styling

.btn__icon.icon__heart

button with an icon (eg.: .icon__heart)

<button class="btn--empty [modifier class]"></button>
<a class="btn--empty [modifier class]"></a>
<input type="radio" class="btn--empty [modifier class]">
